# 数据增强策略优化

本文档描述了对前列腺多模态分割项目数据增强策略的优化，包括新增的增强方法、配置选项和使用指南。

## 优化概述

我们优化了数据增强策略，提供了更多医学影像特定的增强方法，支持自适应增强强度调整，并增加了多种增强策略选项。这些优化旨在提高模型的泛化能力，特别是在数据量有限的情况下。

## 主要改进

### 1. 扩展的TorchIO增强策略

我们在`src/torchio_transforms.py`中扩展了数据增强方法，添加了以下医学影像特定的增强技术：

- **运动伪影 (RandomMotion)**: 模拟MRI扫描过程中患者运动产生的伪影
- **偏置场 (RandomBiasField)**: 模拟MRI中的磁场不均匀性
- **强度缩放 (RandomIntensityScaling)**: 调整图像强度范围
- **Gamma变换 (RandomGamma)**: 应用非线性强度变换
- **区域交换 (RandomSwap)**: 随机交换图像中的区域
- **各向异性 (RandomAnisotropy)**: 模拟不同方向上的分辨率差异

### 2. 新增增强策略

我们添加了多种增强策略，可根据不同场景选择：

- **标准策略 (standard)**: 包含几何变换和强度变换的全面增强
- **轻量级策略 (light)**: 适用于数据量较小或模型容易过拟合的情况
- **仅强度变换 (intensity_only)**: 只应用强度变换，不包含几何变换
- **自适应策略 (adaptive)**: 增强强度随训练进度动态调整

### 3. 自适应增强强度调整

实现了自适应增强强度调整功能，使增强强度随训练进度变化：

- 训练初期使用较强的增强，帮助模型学习更鲁棒的特征
- 随着训练进行，逐渐降低增强强度，使模型在后期稳定训练
- 支持自定义初始强度、最终强度和衰减率

### 4. 增强的传统数据增强方法

在`src/transforms3d.py`中添加了更多传统数据增强方法：

- **随机旋转**: 支持任意角度和轴的旋转
- **随机缩放**: 支持随机缩放和尺寸调整
- **强度偏移**: 随机调整图像强度
- **Gamma校正**: 应用非线性强度变换
- **噪声添加**: 支持高斯、泊松和斑点噪声
- **边界感知弹性变形**: 在边界区域应用更强的变形

### 5. 配置文件更新

在`config.yml`中添加了详细的数据增强配置选项，包括：

- 增强策略选择
- 几何变换参数
- 弹性形变参数
- 医学影像特定增强参数
- 强度变换参数
- 自适应增强参数

## 使用方法

### 1. 配置增强策略

在`config.yml`中配置数据增强策略：

```yaml
# 数据增强配置
augmentation:
  # 增强策略: 'standard', 'light', 'intensity_only', 'adaptive'
  strategy: 'adaptive'
  
  # 几何变换参数
  flip_axes: [0, 1, 2]
  flip_probability: 0.5
  
  # 仿射变换参数
  affine:
    degrees: 15
    translation: 10
    scales: 0.15
    probability: 0.7
  
  # 自适应增强参数
  adaptive:
    initial_intensity: 1.0
    final_intensity: 0.5
    decay_rate: 0.5
```

### 2. 训练中使用

训练脚本会自动使用配置文件中的增强设置。对于自适应增强，需要在每个epoch开始时更新数据集的epoch值：

```python
# 在训练循环中
for epoch in range(num_epochs):
    # 更新数据集的epoch，用于自适应增强
    train_dataset.set_epoch(epoch)
    
    # 训练代码...
```

### 3. 自定义增强配置

可以通过代码直接传递自定义增强配置：

```python
# 自定义增强配置
custom_aug_config = {
    'strategy': 'light',
    'affine': {
        'degrees': 5,
        'translation': 3,
        'scales': 0.05
    }
}

# 创建数据集时传入自定义配置
dataset = ProstateDataset(
    cases=train_cases,
    augment=True,
    augmentation_config=custom_aug_config
)
```

## 增强策略对比

| 策略 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| standard | 通用场景，数据量适中 | 全面的增强，效果好 | 计算开销较大 |
| light | 数据量小，模型易过拟合 | 计算开销小，防止过拟合 | 增强效果有限 |
| intensity_only | 几何变换不敏感的任务 | 保持结构不变 | 几何不变性差 |
| adaptive | 长时间训练，需要鲁棒性 | 动态调整，训练稳定 | 需要调整超参数 |

## 实验建议

1. **数据量较大时**: 使用标准策略，可以充分增加数据多样性
2. **数据量较小时**: 使用轻量级策略，避免过度增强
3. **对几何变换敏感时**: 使用仅强度变换策略
4. **长时间训练时**: 使用自适应策略，可以在训练初期学习鲁棒特征，后期稳定训练

## 性能考虑

1. **计算开销**: 新增的医学影像特定增强方法会增加计算开销，建议根据硬件条件调整增强强度
2. **内存使用**: 某些增强方法（如弹性变形）需要额外内存，建议适当减小batch size
3. **训练时间**: 自适应策略在初期增强强度高，可能延长单次epoch的训练时间
